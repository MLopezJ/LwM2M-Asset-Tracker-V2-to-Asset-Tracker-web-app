# ADR 008: Default LwM2M version

The following versions are expected to be used by the LwM2M objects:

| Object ID | Object Version | LwM2M Version |
| --------- | -------------- | ------------- |
| 3         | 1.2            | 1.1           |
| 4         | 1.3            | 1.1           |
| 6         | 1.0            | 1.0           |
| 3303      | 1.1            | 1.0           |
| 3304      | 1.1            | 1.0           |
| 3323      | 1.1            | 1.0           |

Failure to follow these recommendations could lead to errors.
